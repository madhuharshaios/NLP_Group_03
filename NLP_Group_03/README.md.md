&#x20;Text Summarization System Using Machine Learning and Deep Learning



&#x20;Project Description

This project is an NLP-based system that generates summaries from news articles using ML and DL models.



&#x20;Dataset

News Summary Dataset (Kaggle)

[https://www.kaggle.com/datasets/sunnysai12345/news-summary](https://www.kaggle.com/datasets/sunnysai12345/news-summary)



&#x20;Models Used

\- Logistic Regression

\- Naive Bayes

\- SVM

\- CNN

\- LSTM

\- GRU



&#x20;How to Run

pip install -r requirements.txt

streamlit run app.py
